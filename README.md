# Smart Travel Project Documentation

## Project Overview
Smart Travel is an innovative project designed to offer travelers a convenient and secure way to access various services and resources while on the road.

## Features by Category
### Fuel & Roadside Help
- Find nearby fuel stations
- Roadside assistance contacts
- Emergency vehicle services

### Lodges & Accommodation
- List of nearby lodges
- Booking information
- Review options

### Markets & Shops
- Directory of local markets
- Shop reviews
- Operating hours of shops

### Mechanics & Repairs
- List of nearby mechanics
- Service reviews
- Repair guides

### AI/IDS Security System
- Integrated AI for real-time alerts
- Intrusion Detection System (IDS) for enhanced security

## Tech Stack
- Flutter Web
- Firebase
- Firestore

## Project Structure
- `lib/`: Contains the main application code
- `assets/`: Contains images and other static resources
- `test/`: Contains unit tests

## Getting Started Guide
1. **Clone the repository**: `git clone https://github.com/aubreykumwenda/smart-travel.git`
2. **Change directory**: `cd smart-travel`
3. **Install dependencies**: `flutter pub get`
4. **Run the project**: `flutter run`

## Firebase Setup Instructions
1. **Create a Firebase project**: Go to the Firebase console and create a new project.
2. **Add your app**: Follow the instructions to register your app with Firebase.
3. **Download the configuration file**: Save it in your project folder as `google-services.json` for Android or `GoogleService-Info.plist` for iOS.
4. **Enable Firestore**: Set up Firestore in your Firebase project.
